
# House Price Prediction

This project focuses on predicting house prices using machine learning techniques. 
The objective is to estimate the price of a house using features such as area, number of bedrooms, bathrooms, stories, and parking.

The project also explores feature importance to understand which variables have the strongest influence on house prices.

## Technologies
- Python
- Scikit-learn
- Pandas
- Matplotlib
- Seaborn

## Model
A Random Forest model with hyperparameter tuning using GridSearchCV was used to improve prediction performance.
